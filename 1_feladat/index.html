Táblázat menü

Dinamikus táblázat CRUD (Create, Read, Update, Delete) műveletekkel, valós idejű validációval minden mezőre (kötelező kitöltés, min/max hossz). A táblázat minden oszlopa alapján szűrhető és rendezhető.

HTML5 menü

Rövid, szemléltető példák a HTML5 újdonságaiból: Web Storage, Web Workers, Server-Sent Events, Geolocation API, Drag and Drop API, Canvas és SVG. A példák külön-külön, vagy egy oldalon belül almenüként is elérhetőek.

ChartJS menü

Egy 5x5-ös számokat tartalmazó táblázatból választható sorok adatainak vizuális megjelenítése vonaldiagram segítségével a ChartJS könyvtár segítségével.

AJAX menü

AJAX-alapú alkalmazás egy API-val való kommunikációra. CRUD műveletek végrehajtása validációval, adatok kiírása, statisztikák számítása (összeg, átlag, legnagyobb érték) és sikerességi visszajelzések megjelenítése.

OOJS menü

Objektumorientált JavaScript alkalmazás példája, ahol class, constructor, metódusok, öröklődés (extends, super) és a DOM manipuláció (appendChild) használata is bemutatásra kerül.


Hasznos információk

  • Modern webfejlesztési technológiák bemutatása
  • Gyakorlati példák interaktív táblázatokra
  • HTML5 API-k alkalmazása
  • Valós idejű adatszolgáltatás AJAX segítségével
  • Objektumorientált szemlélet JavaScript-ben
